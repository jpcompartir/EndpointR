# Retrieve information about an endpoint

Retrieve information about an endpoint

## Usage

``` r
hf_get_endpoint_info(endpoint_url, key_name = "HF_API_KEY")
```

## Arguments

- endpoint_url:

  Hugging Face Embedding Endpoint

- key_name:

  Name of environment variable containing the API key (default:
  "HF_API_KEY")

## Value

JSON of endpoint information
